### Start ZooKeeper

```powershell
docker run-d--name zookeeper-p2181:2181-e ZOOKEEPER_CLIENT_PORT=2181-e ZOOKEEPER_TICK_TIME=2000 confluentinc/cp-zookeeper:7.5.0

```

### Start Kafka

```powershell
docker run -d --name kafka -p 9092:9092 -e KAFKA_BROKER_ID=1 -e KAFKA_ZOOKEEPER_CONNECT=zookeeper:2181 -e KAFKA_ADVERTISED_LISTENERS=PLAINTEXT://localhost:9092 -e KAFKA_OFFSETS_TOPIC_REPLICATION_FACTOR=1 --link zookeeper confluentinc/cp-kafka:7.5.0
```

docker exec -it kafka bash

create topic
kafka-topics --create --topic my-topic --bootstrap-server localhost:9092 --partitions 3 --replication-factor 1


consumer
read messages via topic with group (optional)
kafka-console-consumer --topic my-topic --bootstrap-server localhost:9092 --group group-A
